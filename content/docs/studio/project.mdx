---
title: Project
description: An overview of the Project page in the Agentsmith Studio.
---

The project page contains holistic information about your project.

- Information on your GitHub repository connection and GitHub App installation
- The last time a sync was performed with a "Sync Project" button to sync again
- The ability to download your types manually via the "Download Types" button
- A list of your prompts for quick access
- Instructions on how to get started with the Agentsmith SDK
- A reference to your global variables
