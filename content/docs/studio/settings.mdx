title: Settings
description: An overview of the Settings page in the Agentsmith Studio.
---

The Settings page is where you can manage your organization's integrations and access your API key.

![Settings Page Screenshot](/studio-screenshots/settings.png)

From this page you can:

- **GitHub App Installation**: Connect your Agentsmith organization to a GitHub account. This is required for syncing your prompts with a GitHub repository. You can also see which repositories are accessible and connect them to your projects.
- **Openrouter Account**: Connect your Agentsmith organization to an Openrouter account to use their LLMs.
- **API Key**: View and copy your organization's API key. This key is used to authenticate with the Agentsmith API.
